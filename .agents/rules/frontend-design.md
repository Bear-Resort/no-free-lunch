# Frontend design (agent rule)

Apply when working on UI, components, theme, or styles.

- Stack: React, Tailwind, shadcn/ui
- Theme: dark blue tokens from `docs/design-system.md`
- Modals: `backdrop-blur` + dimmed overlay; moderate open/close animation
- Loading: skeletons or pending buttons on every Convex-driven UI
- Prefer `$nfl-frontend` skill for implementation details
